Installation Guide · Windows

Home Assistant MCP
+ Claude Code

Connect your HA instance to Claude Code in VS Code via MCP

Prerequisites — install these first
⬡
Node.js (LTS) nodejs.org
⬛
VS Code code.visualstudio.com
✦
Claude Code VS Code Marketplace


Steps
1
Enable MCP in Home Assistant
  1. In HA go to Settings → Devices & Services → Add Integration
  2. Search for and install Model Context Protocol Server
  3. Go to your Profile → Long-Lived Access Tokens
  4. Click Create Token, name it claude-code, and copy it immediately
2
Verify your token works

In PowerShell, run:

powershell 
curl -H "Authorization: Bearer YOUR_TOKEN" http://YOUR_HA_IP:8123/api/
✓ You should get {"message":"API running."} back. Use the raw IP address, not the hostname.
3
Open your HA repo in VS Code

Open the folder where your HA config is stored (your local GitLab/GitHub repo).

4
Add the MCP server via Claude Code

Open the Claude Code panel in VS Code, then in its terminal run:

powershell 
claude mcp add-json homeassistant '{
  "type": "stdio",
  "command": "npx",
  "args": ["-y", "hass-mcp"],
  "env": {
    "HASS_URL": "http://YOUR_HA_IP:8123",
    "HASS_TOKEN": "YOUR_FULL_TOKEN_HERE"
  }
}' -s local
⚠ Use the raw IP address, not homeassistant.local. The -s local flag scopes this config to the current project only.
5
Verify connection

In Claude Code, type:

claude code 
/mcp
✓ You should see homeassistant · ✔ connected
6
Protect your token

Add the following to your .gitignore:

.gitignore 
.vscode/mcp.json
⚠ Never commit .claude.json either — it contains your token in plain text.
7
Test it

In Claude Code, try asking:

prompt 
use my MCP to list home assistant areas
✓ You should get a live table of your HA areas pulled directly from the running instance.


Lessons learned
✕ Don't use https:// for local HA — use http://
✕ Don't use the OAuth approach in .vscode/mcp.json — it doesn't work for local HA
✕ Don't use HA_URL / HA_TOKEN — the package requires HASS_URL / HASS_TOKEN
✕ Don't use the hostname — use the raw IP for subprocess resolution on Windows
✓ Use claude mcp add-json with -s local, not the VS Code mcp.json file
